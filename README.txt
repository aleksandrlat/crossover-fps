Test project.

This application allows to connect patients and blood donors around the world.

Requirements.
- nodejs. I used for development v6.6.0
- npm. I used installed with nodejs v6.6.0
- mongodb. I used v3.2.9

Instructions.
- start mongodb on localhost default port
- cd to Code folder
- run nmp install
- run npm run build
- run npm run start
- navigate to http://localhost:3000/ in browser

To run tests.
- cd to Tests folder
- run nmp install
- run npm run test
